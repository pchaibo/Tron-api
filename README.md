
提供波场TRON trc20支付接口代码，扫区块通知回调api
===============
> 1.运行环境要求PHP7.4
> 2.安装gmp扩展，否则转账不成功
> 3.配置项目伪静态
> 4.合约地址不需要动，切记别改
> 5.ThinkPHP基本运行要求
## 目前功能
* 生成钱包地址
* 获取USDT余额
* 获取TRX余额
* 资金自动归集
* 查询订单详情
* 私钥获取地址
* 查询最新区块
* 根据区块链查询信息
* USDT交易转账
* TRX交易转账（可添加备注）
* 查询最新交易（USDT）
